# Portfolio

A static portfolio website built with React and Vite.

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Project structure

```text
src/
├── app/          Application shell
├── components/   Reusable UI and layout components
├── sections/     Homepage sections
└── styles/       Global styles and design tokens
```
