# Images

Store portfolio images that are imported by React components in this directory.
Vite will process, optimize, and fingerprint these files during production builds.

Use descriptive, lowercase kebab-case filenames, for example:

```text
screen-magnifier-hero.webp
campus-restaurant-api-dashboard.webp
qi-pan-portrait.webp
```

Prefer WebP or AVIF for photographs and PNG only when transparency is required.

Import an image from a component like this:

```jsx
import projectImage from '../assets/images/screen-magnifier-hero.webp'
```
